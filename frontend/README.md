# Frontend - AI Ticket Dashboard

Next.js 15 App Router dashboard for AI support ticket classification.

## Stack
- Next.js 15 + TypeScript
- Tailwind CSS + shadcn/ui-style components
- Framer Motion animations
- Axios API layer
- Dark mode with `next-themes`

## Run
```bash
npm install
npm run dev
```

Set environment:
```env
NEXT_PUBLIC_API_URL=http://localhost:8000
```
